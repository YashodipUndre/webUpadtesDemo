# WebUpdates

This is a Next.js project with a premium landing page design.

## Improvements Made
- **Fixed Dependency Issue**: Manually installed `lightningcss-win32-x64-msvc` to resolve the start-up error.
- **Premium Design**: Updated `src/app/page.tsx` with a responsive, modern UI.

## How to Run

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
-   `src/app/page.tsx`: Main landing page using Tailwind CSS v4.
-   `src/app/globals.css`: Global styles.
